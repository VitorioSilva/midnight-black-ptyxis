# Midnight Black

Tema escuro personalizado para o **Ptyxis**, o terminal padrão do Ubuntu 26.04 LTS.

Inspirado no visual limpo do macOS Terminal.app, com fundo preto puro, texto branco, e uma barra de título em cinza escuro. Projetado para longas sessões de código sem cansar a vista.

---

## Paleta de Cores (Dark Mode)

| Cor | Hex | Uso |
|-----|-----|-----|
| Background | `#000000` | Fundo preto puro |
| Foreground | `#FFFFFF` | Texto branco puro |
| Titlebar | `#303030` | Barra superior cinza escuro |
| Cursor | `#FFFFFF` | Cursor branco |
| Color8 | `#8E8E93` | Comentários e texto secundário |
| Red | `#FF5F57` | Erros |
| Green | `#28C840` | Sucesso |
| Yellow | `#FFBD2E` | Avisos |
| Blue | `#007AFF` | Links e diretórios |
| Magenta | `#AF52DE` | Destaque secundário |
| Cyan | `#5AC8FA` | Informações |

---

## Instalação

1. Clone o repositório ou baixe o arquivo `midnight-black.palette`
2. Copie para o diretório de paletas do Ptyxis:

```bash
cp midnight-black.palette ~/.local/share/org.gnome.Ptyxis/palettes/
